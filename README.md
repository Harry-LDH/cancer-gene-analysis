# Cancer Gene Frequencies Analysis in Chromosome 8 Across Populations

## Overview
In a team of 3, we used Python and R with various libraries and made an analysis of chromosome 8 data across each group of global populations to study which genes carry the highest pathogenic mutation rates, and which population carries them by how much.

## Data Source
- ClinVar: NIH clinical genomic database
- gnomAD: Genome Aggregation Database (chromosome 8)

## Key Findings
- RECQL4, WRN, NBN, and EXT1 had the highest rates of pathogenic mutations in chromosome 8.
- Non-Finnish Europeans were found to have the highest rate of pathogenic cancer gene mutations in RECQL4 and WRN.
- Certain cancer genes may be more prevalent or have higher frequencies in African and American populations.
